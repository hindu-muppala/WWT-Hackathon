## Overview

This repository contains solutions and prototypes submitted for the WWT All India Women Hackathon 2024. It demonstrates innovative approaches for agricultural data analysis and demand forecasting using Jupyter Notebooks and Python.

## Repository Structure

- **AgriHelper/**  
  Tools and scripts to assist farmers with data-driven decisions, including crop recommendations and agricultural guidance.

- **CropDemandFocast/**  
  Modules for forecasting crop demands using historical data, machine learning models, and analytics.

- **README.md**  
  Project description and directory index.

## Technologies Used

- **Jupyter Notebook** (main implementation)
- **Python** (for data scripting & analytics)

## Instructions

1. Clone the repository
2. Explore subfolders for individual solutions
3. Launch and run Jupyter Notebooks to view code and results
